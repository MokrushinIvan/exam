# Платформер на JavaScript

Простой, но полностью рабочий 2D-платформер с физикой, коллизиями, сбором монет и условиями победы/поражения.

## Использованные нейросети
- **Grok 4 (xAI)** — основная генерация всей архитектуры и кода (~80%)
- **Cursor** — доработка и отладка (~20%)

**Время выполнения:** ~3.5 часа

## Функционал
- Управление: ←→ / A D — движение, Пробел / W — прыжок
- Гравитация и физика
- Коллизии с платформами
- Сбор монет (+10 очков)
- Камера следует за игроком
- Условие победы (все монеты собраны)
- Поражение при падении

## Запуск
Просто откройте `index.html` в браузере или используйте GitHub Pages.
